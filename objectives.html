Objective

The objective of this app is to allow its users to sign in and allow it to scan class syllabi to auto-populate important dates and assignments onto a calendar of their choosing. Also allows for simple push notifications that can remind users of important dates. The main goal of the app is to allow for fast and simple organization.

Concerns for Mobile App

  • Since the app is new our concerns are that the customers would prefer to use their accustomed Calendar (e.g., Google Calendar, Apple Calendar, or Outlook Calendar)
  • UI and visual design would not display nicely on all phone types such as Android and Apple so using one set of guideline principles would not translate well to another operating system.
  • App will be able to use system accessibility features so any needs will be met via device preferences.

Ideas

  • Allow user to customize app appearance: User can set appearance to one of the color palette choices we provide also having different accessibility options; Once the user picks the palatte of their choice the app will refresh and their desired colors will appear.
  • Partnership with education technology companies such as MyCourses so that it is known that our application can easily pair syllabus information. Possibly have it so that along with the syllabus is an ad to download our application.
  • Accessible on mobile and desktop devices. The app is responsive and works properly on various devices.
  • Just in case the parsing software doesn't find correct or proper dates have an option to manually put the date in as well.
