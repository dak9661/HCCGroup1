Stakeholder Goals: For individuals to efficently scan information and dates into a calendar

  • Secure application - this is important for students, professors and all users
  • Appealing appearance - impoarnt for all users so it can be user friendly and simple to navigate
  • Information is scanned and processed in a timely fashion - ensuring that it doesnt take too much time to scan or else it would be useless
  • Simple to use with featues making it accessible to any potential user - so both parents, professors and students will easily be able to use it
